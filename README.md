ABOUT<br />
LockSmith is a lightweight, offline and easy to use tool that lets
you generate and store passwords for all of your accounts.

Password generator is available as a standalone program. Main project is currently a work-in-progress so it is not recommended to be used as a legitimate password manager at this point. More info can be found in releases.

Addtitonal credits to https://github.com/dwyl/english-words for the list of words used to generate passwords.
