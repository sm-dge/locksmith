ABOUT<br />
LockSmith is a lightweight, offline and easy to use tool that lets
you generate and store passwords for all of your accounts.

Password generator is available as a standalone program. More info about the main project can be found in releases. Currently a work-in-progress. Main file is not an executable so it requires Python as well as the Pandas and Cryptography modules to run.

Addtitonal credits to https://github.com/dwyl/english-words for the list of words used to generate passwords.
